# Puerto Rico Travel Chatbot

AI-powered travel assistant focused on Puerto Rico tourism using Streamlit, LangChain, FAISS, OpenAI embeddings, and weather API integration.

## Features
- Personalized travel recommendations
- Retrieval-Augmented Generation (RAG)
- Weather integration
- Interactive chatbot interface
- Tourism-focused itinerary assistance

## Technologies Used
- Python
- Streamlit
- LangChain
- FAISS
- OpenAI API
- BeautifulSoup
